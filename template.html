Overview: Go-No Go is a game where you try and respond correctly to the signal given by the big button, If you see the green button which says "Go!" you want to click it within a limited amount of time. If you see the red button which says "No Go!" do not click it and wait until the next signal.
Statistics: Correct and incorrect responses, as well as reaction time, are tracked by the app and can be seen after playing the game or by tapping "Statistics" on the main menu screen. The main statistics screen shows stats across all previous games. From this screen, tapping on a previous game will pull up more detailed statistics. Clicking the button in the top right corner of the statistics screen will give users the option to delete all data.
Modes: There are two game modes: Standard and Endless. In Standard mode there are always 30 signals given. Incorrect responses will not end the game. In Endless mode the game continues until there is an incorrect response. Each correct response counts as a point.
